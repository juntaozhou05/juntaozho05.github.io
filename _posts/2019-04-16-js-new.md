### 一：new操作符做了什么

1. 创建一个新对象；

2. 将构造函数的作用域赋给新对象（因此 this 就指向了这个新对象） ；

3. 执行构造函数中的代码（为这个新对象添加属性） ；

4. 返回新对象（如果构造函数返回的是一个对象，那么new出来的就是该对象）。

### 二：自己实现一个new方法

```
var new2 = function(fun){
    var o = new Object()
    o.__proto__ = fun.prototype
    var k = fun.call(o)
    if(typeof k === 'object') {
    	return k
    }else {
	return o
    }
}
```
